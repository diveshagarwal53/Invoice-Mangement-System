# 🚀 Invoice Management System (SaaS Style)

A full-stack Invoice Management System built using modern web technologies. This project allows users to manage clients, create invoices, and track revenue with an interactive dashboard.

---

## ✨ Features

* 🔐 Authentication (Login / Protected Routes)
* 📄 Invoice CRUD (Create, Read, Update, Delete)
* 👥 Client-based invoice management
* 🔍 Search & Pagination
* 📊 Dashboard Analytics

  * Total Revenue
  * Paid / Unpaid / Overdue stats
  * Monthly Revenue Chart 📈
* 🎨 Modern UI with Tailwind CSS
* ⚡ Responsive Design

---

## 🛠️ Tech Stack

### Frontend

* React.js
* Tailwind CSS
* Recharts (for charts)

### Backend

* Node.js
* Express.js

### Database

* MongoDB (Mongoose)

### Other Tools

* JWT Authentication
* Git & GitHub

## ⚙️ Installation & Setup

### 1️⃣ Clone the repository

```bash
git clone https://github.com/yourusername/invoice-management-system.git
cd invoice-management-system
```

---

### 2️⃣ Install dependencies

#### Backend

```bash
cd backend
npm install
```

#### Frontend

```bash
cd frontend
npm install
```

---

### 3️⃣ Setup Environment Variables

Create a `.env` file in backend:

```env
PORT=7000
MONGO_URI=your_mongodb_connection
JWT_SECRET=your_secret_key
```

---

### 4️⃣ Run the project

#### Backend

```bash
npm run dev
```

#### Frontend

```bash
npm run dev
```
